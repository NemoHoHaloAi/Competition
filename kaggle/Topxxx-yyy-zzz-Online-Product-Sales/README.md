# README

- [项目链接](https://www.kaggle.com/c/online-sales/data)
- 项目目的：构建模型，尽可能好的预测在线商品的月销售情况；
- 数据：
    - 格式：csv
    - 大小：4M
    - 每一行表示一个不同的消费品；
    - 前12列（Outcome_M1~Outcome_M12）表示商品上市后的1~12个月的销售情况；
    - Date_1表示
    - Date_2表示
    - Quan_x表示定量特征；
    - Cat_x表示定性特征；
    - 数据有缺失MISSING：Quan_5、Outcome_x；
- RMSLE (root mean square logarithmic error)；
